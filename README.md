# Network Scanner (C#)

This is a simple network scanner I built to practice basic networking concepts in C#.

The idea is straightforward:  
you enter an IP range, and the program checks which devices are active using ping.

## What it does
- Scans a range of IP addresses
- Shows only active devices
- Displays response time (latency)

## Why I built this
I wanted to better understand how network connectivity works and get some hands-on experience instead of just learning theory.

## Technologies
- C#
- .NET
- Ping (System.Net.NetworkInformation)

## Notes
This is a simple tool, but it helped me understand:
- how ping works
- how to handle network responses
- basic parallel processing in C#
